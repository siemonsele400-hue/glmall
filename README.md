# 谷粒商城 - 分布式电商项目（SpringBoot3 + Vue3）

个人独立手敲的完整分布式电商系统，持续更新中……

## 项目介绍
- 基于 SpringBoot 3 + MyBatis-Plus + Redis + MySQL8 + Docker
- 包含商品、订单、仓储、支付、秒杀等完整业务模块
- 前后端分离，后端全部由我独立完成，前端进行中

## 技术栈
- SpringBoot 3.x / MyBatis-Plus / Redis / MySQL8
- RabbitMQ / Elasticsearch / Docker / Nginx
- Vue3 + Element-Plus + Vite（前端进行中）
